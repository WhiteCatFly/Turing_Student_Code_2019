## Usage
```
make
./crawl -h
-u <url>	Set the root url to crawl
		DEFAULT=http://info.ruc.edu.cn/

-s <path>	Set the path to save save_file
		DEFAULT=crawl.save

-load <path>	Continue crawling with the save_file

-o <path>	Redirection web page save location
		DEFAULT=page/
```
