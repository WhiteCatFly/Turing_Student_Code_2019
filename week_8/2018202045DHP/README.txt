Search Engine version 1.2

Before using this program, you need to download THULAC to support some of the functions.
THULAC folder should be put together with all of these files.

User hint:
make
./IW
./SE

A very naive search engine, with modifying twice.

Use tf-idf to evaluate the value of page.
Page (at most 10 pages) with higher value will be presented at top.

Support phrase (it will be cut into words) searching.
